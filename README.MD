# TaskFlow

## Problem

Students and professionals often struggle to organize their daily tasks, meet deadlines, and track their progress. Keeping everything in one place can be difficult, which leads to missed tasks and reduced productivity.

## Solution

TaskFlow is a task management application that helps users organize, manage, and track their daily tasks and long-term goals.

## Features

- Add New Task
- Edit Task
- Delete Task
- Search Tasks
- Mark Tasks as Completed
- View Pending Tasks
- Track Progress
- Set Deadlines
- Task Reminders
